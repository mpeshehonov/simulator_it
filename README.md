# DevLife 8-bit

Карьерный симулятор разработчика — Telegram Mini App.

## Запуск

```bash
npm install
npm run dev
```

## Скрипты

- `npm run dev` — разработка
- `npm run build` — сборка
- `npm run start` — production
- `npm run lint` — ESLint
- `npm run format` — Prettier

## Окружение

Скопируй `.env.local.example` в `.env.local` и заполни переменные.

## Структура

См. `docs/CONCEPT.md` — концепция и механики. `docs/MANUAL_STEPS.md` — что сделать вручную.
